Q. What is the $_SERVER["PHP_SELF"] variable?
>> The $_SERVER["PHP_SELF"] is a super global variable that returns the filename of the currently executing script.

Q. What is the htmlspecialchars() function?
>> The htmlspecialchars() function converts special characters into HTML entities. This means that it will convert HTML characters like < and > with &lt; and &gt;. This prevents attackers from exploiting the code by injecting HTML or Javascript code (Cross-site Scripting attacks) in forms.
